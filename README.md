# sql-database-practice
SQL practice covering database design, table creation and basic queries for structured data management.

## Description
The project demonstrates how to create database tables, insert records,
and perform queries for reporting and analysis.

## Topics Covered
- Table creation and schema design
- Data insertion
- Filtering and aggregation queries
- Basic reporting with SQL

## Tools & Technologies
- SQL
- SQLite / MySQL (conceptual)

## Goal
To strengthen SQL fundamentals and database management skills
required for data engineering and data systems roles.
